# DevOps

A simple Express.js "Hello World" server.

## Requirements

- Node.js (v18 or higher)
- npm

## Installation

```bash
npm install
```

## Run

```bash
npm start
```

The server runs on port `3000` by default. To use a custom port, set the `PORT` environment variable:

```bash
PORT=8080 npm start
```

Open `http://localhost:3000` in your browser to see "Hello World".
